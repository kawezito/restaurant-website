## Sobre

Projeto desenvolvido para a disciplina de Introdução às Tecnologias Web.

A proposta foi criar um site de restaurante aplicando conceitos fundamentais como HTML semântico, CSS responsivo e manipulação básica com JavaScript.

## Acesso

https://restaurant-website-ten-drab.vercel.app/

## Funcionalidades

* Layout responsivo
* Menu mobile (hamburguer)
* Slider automático
* Cards gerados via JavaScript
* Filtro por categorias
* Formulário com validação
* Mapa integrado

## Tecnologias

* HTML
* CSS
* JavaScript

## Estrutura

```bash
src/
├── index.html
├── style.css
└── script.js
```

## Execução

```bash
git clone https://github.com/kawezito/restaurant-website.git
cd restaurant-website
```

Abrir `src/index.html` no navegador.
